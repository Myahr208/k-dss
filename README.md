# k-dss

### dependencies
* `nodejs` V8 or higher

### build
```sh
git clone git@github.com:dapphub/k-dss.git
make all
```

### usage
to run a proof with [klab](https://github.com/dapphub/klab):
```sh
klab run --spec out/frob_success_file_bytes32_bytes32_int256.ini
```

or any other spec in the `out` dir
